rgpycrumbs.chemgp.plotting¶

ChemGP plotting functions.

This module provides pure plotting functions for ChemGP data visualization. All functions are format-agnostic and delegate to chemparseplot for actual plotting. No file I/O, no CLI logic - pure functions only.

Added in version 1.7.0: Extracted from chemgp.plt_gp to standalone module.
