

ChemGP Plotting
---------------

These tools generate publication figures from
`ChemGP <https://github.com/HaoZeke/ChemGP>`_ HDF5 output files, using
`chemparseplot <https://chemparseplot.rgoswami.me>`_ for visualization.

Performance notes
~~~~~~~~~~~~~~~~~

- Parallel batch processing with ``--parallel/-j`` flag (3-5x speedup)

- Caching with ``@lru_cache`` for repeated operations

- Complete type hints and HDF5 validation
