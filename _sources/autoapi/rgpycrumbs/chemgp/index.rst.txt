rgpycrumbs.chemgp
=================

.. py:module:: rgpycrumbs.chemgp

.. autoapi-nested-parse::

   ChemGP plotting and I/O utilities.

   This module provides functions for reading ChemGP HDF5 data and generating
   publication-quality plots.

   .. versionadded:: 1.7.0
       Refactored from monolithic plt_gp.py to modular structure.

   Examples
   --------
   Read HDF5 data and plot:

   >>> import h5py
   >>> from rgpycrumbs.chemgp import read_h5_grid, plot_surface
   >>> with h5py.File("data.h5", "r") as f:
   ...     data, xc, yc = read_h5_grid(f, "energy")
   ...     gx, gy = np.meshgrid(xc, yc)
   ...     plot_surface(gx, gy, data, "output.pdf")
