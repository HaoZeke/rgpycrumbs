#+TITLE: NEB Reaction Path Visualization
#+OPTIONS: toc:nil todo:nil num:nil author:nil title:nil

#+begin_export rst
=============================
NEB Reaction Path Visualization
=============================
#+end_export

In this tutorial you will produce publication-quality figures from a
Diels-Alder NEB calculation: a 1D energy profile, a 2D reaction
valley landscape with GP surface, structure galleries with four
rendering backends, OCI-NEB peak overlays, and band evolution plots.

* Prerequisites

#+begin_src bash
# Core package with all rendering backends
pip install "rgpycrumbs[all]" xyzrender solvis-tools ovito
#+end_src

The tutorial data ships with the rgpycrumbs source tree at
=docs/orgmode/tutorials/data/diels_alder/=. It contains output from an
OCI-NEB calculation of the Diels-Alder [4+2] cycloaddition of
1,3-butadiene and ethylene (C6H10, 16 atoms, 22 images).

* Step 1: Energy Profile

#+begin_src shell
rgpycrumbs eon plt-neb \
  --plot-type profile \
  --input-dat-pattern "data/diels_alder/neb_*.dat" \
  --input-path-pattern "data/diels_alder/neb_path_*.con" \
  --con-file data/diels_alder/neb.con \
  --sp-file data/diels_alder/sp.con \
  -o profile.pdf
#+end_src

Multiple NEB optimization steps are overlaid with Hermite spline
interpolation. The final converged barrier is ~0.8 eV.

* Step 2: 2D Reaction Valley Landscape

** Path overlay (no surface fit)

#+begin_src shell
rgpycrumbs eon plt-neb \
  --plot-type landscape \
  --landscape-mode path \
  --no-project-path \
  --con-file data/diels_alder/neb.con \
  --sp-file data/diels_alder/sp.con \
  -o landscape_raw.pdf
#+end_src

Shows the NEB path on raw RMSD(R,P) axes, colored by energy.

** Projected (s, d) coordinates

#+begin_src shell
rgpycrumbs eon plt-neb \
  --plot-type landscape \
  --landscape-mode path \
  --project-path \
  --con-file data/diels_alder/neb.con \
  --sp-file data/diels_alder/sp.con \
  -o landscape_projected.pdf
#+end_src

The =--project-path= flag rotates into reaction progress (s) and
orthogonal deviation (d). A path with d near 0 indicates a
concerted mechanism.

** GP surface fit

#+begin_src shell
rgpycrumbs eon plt-neb \
  --plot-type landscape \
  --landscape-mode surface \
  --surface-type grad_matern \
  --project-path \
  --con-file data/diels_alder/neb.con \
  --sp-file data/diels_alder/sp.con \
  -o landscape_surface.pdf
#+end_src

Fits a gradient-enhanced Matern kernel GP surface. Shows energy
contours with dashed variance contour lines. Requires jax.

* Step 3: Structure Galleries

** Rendering backends

Four backends are available via =--strip-renderer=:

#+begin_src shell
# xyzrender (default): ball-and-stick with paton preset
rgpycrumbs eon plt-neb \
  --plot-type landscape --landscape-mode path --no-project-path \
  --con-file data/diels_alder/neb.con --sp-file data/diels_alder/sp.con \
  --plot-structures crit_points \
  --strip-renderer xyzrender \
  --strip-spacing 2.0 --strip-dividers \
  -o gallery_xyzrender.pdf

# ASE: space-filling spheres
rgpycrumbs eon plt-neb \
  --plot-type landscape --landscape-mode path --no-project-path \
  --con-file data/diels_alder/neb.con --sp-file data/diels_alder/sp.con \
  --plot-structures crit_points \
  --strip-renderer ase \
  --strip-spacing 2.0 --strip-dividers \
  -o gallery_ase.pdf

# solvis: PyVista ball-and-stick (transparent background)
rgpycrumbs eon plt-neb \
  --plot-type landscape --landscape-mode path --no-project-path \
  --con-file data/diels_alder/neb.con --sp-file data/diels_alder/sp.con \
  --plot-structures crit_points \
  --strip-renderer solvis \
  --strip-spacing 2.0 --strip-dividers \
  -o gallery_solvis.pdf

# ovito: OVITO off-screen rendering
rgpycrumbs eon plt-neb \
  --plot-type landscape --landscape-mode path --no-project-path \
  --con-file data/diels_alder/neb.con --sp-file data/diels_alder/sp.con \
  --plot-structures crit_points \
  --strip-renderer ovito \
  --strip-spacing 2.0 --strip-dividers \
  -o gallery_ovito.pdf
#+end_src

** Viewing angle

The =--rotation= flag controls the viewing angle for all backends:

#+begin_src shell
# Default: 0x, 90y, 0z (side view)
rgpycrumbs eon plt-neb ... --rotation "0x,90y,0z"

# Top-down view
rgpycrumbs eon plt-neb ... --rotation "90x,0y,0z"

# Custom angle
rgpycrumbs eon plt-neb ... --rotation "45x,30y,0z"
#+end_src

** Perspective tilt

When atoms overlap in orthographic projection, use =--perspective-tilt=
to apply a small Rodrigues off-axis rotation that reveals hidden atoms:

#+begin_src shell
# 8 degrees of tilt reveals H atoms hidden behind the ring
rgpycrumbs eon plt-neb \
  --plot-type landscape --landscape-mode path --no-project-path \
  --con-file data/diels_alder/neb.con --sp-file data/diels_alder/sp.con \
  --plot-structures crit_points \
  --strip-dividers \
  --perspective-tilt 8 \
  -o gallery_tilted.pdf
#+end_src

* Step 4: OCI-NEB Peak Overlay

Overlay MMF (mode-following) dimer refinement peaks:

#+begin_src shell
rgpycrumbs eon plt-neb \
  --plot-type landscape --landscape-mode path --no-project-path \
  --con-file data/diels_alder/neb.con --sp-file data/diels_alder/sp.con \
  --mmf-peaks --peak-dir data/diels_alder/ \
  -o mmf_peaks.pdf
#+end_src

Star markers show where the dimer refinement placed the saddle.

* Step 5: Band Evolution

Show how the NEB band evolved during optimization:

#+begin_src shell
rgpycrumbs eon plt-neb \
  --plot-type landscape --landscape-mode path --no-project-path \
  --con-file data/diels_alder/neb.con \
  --show-evolution \
  -o evolution.pdf
#+end_src

Older bands are drawn with lower opacity.

* Summary

| Feature | Flag | Notes |
|---------+------+-------|
| Energy profile | =--plot-type profile= | Hermite spline |
| Raw RMSD landscape | =--no-project-path= | RMSD(R) vs RMSD(P) |
| Reaction valley | =--project-path= | (s, d) coordinates |
| GP surface | =--landscape-mode surface= | Needs jax |
| Structure strip | =--plot-structures crit_points= | R, SP, P |
| All structures | =--plot-structures all= | All images |
| xyzrender | =--strip-renderer xyzrender= | Default, ball-and-stick |
| ASE | =--strip-renderer ase= | Space-filling |
| solvis | =--strip-renderer solvis= | PyVista, transparent bg |
| ovito | =--strip-renderer ovito= | OVITO off-screen |
| Custom rotation | =--rotation "45x,30y,0z"= | All backends |
| Perspective tilt | =--perspective-tilt 8= | Reveals hidden atoms |
| Spacing/dividers | =--strip-spacing 2.0 --strip-dividers= | |
| MMF peaks | =--mmf-peaks --peak-dir .= | OCI-NEB |
| Band evolution | =--show-evolution= | Fading iterations |
