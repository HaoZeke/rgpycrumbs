* Hub (~rgpycrumbs.api~)

Prefer::

#+BEGIN_SRC python
from rgpycrumbs.api import (
    load_config,
    load_pypi_pins,
    apply_pin_to_spec,
    pins_from_env,
    ensure_import,
    resolve_lock_path_layered,
    user_config_path,
)
#+END_SRC

| Symbol | Role |
|--------+------|
| ~load_config~ | Merge ~/~/.config/rgpkgs/~ + project ~rgpkgs.toml~ |
| ~load_pypi_pins~ | Parse uv.lock / pylock / CycloneDX → pin map |
| ~apply_pin_to_spec~ | Apply pin map to a pip requirement string |
| ~pins_from_env~ | Read ~RGPKGS_LOCK_PINS~ (set by CLI dispatch) |
| ~ensure_import~ | On-demand optional import (+ AUTO_DEPS) |
| ~resolve_lock_path_layered~ | CLI > env > config lock path |
| ~user_config_path~ | Path to global TOML |

CLI debug: ~rgpycrumbs config show~

* chemparseplot (~chemparseplot.api~)

Bare install; extras transitional. Prefer ~chemparseplot.api~ for new consumers.

** Representative library path: parse → typed result

#+BEGIN_SRC python
from chemparseplot.api import extract_orca_geomscan_energy

dist_bohr, energy_Eh = extract_orca_geomscan_energy(orca_out_text, "Actual Energy")
# pint Quantities: distance (bohr), energy (hartree); non-empty on real scan output
assert dist_bohr.size > 0
assert str(energy_Eh.units) == "hartree"
#+END_SRC

Also re-exported:

| Symbol | Role |
|--------+------|
| ~extract_orca_geomscan_energy~ | ORCA geomscan surface → ~(Q_ bohr, Q_ hartree)~ |
| ~convert_energy_magnitude~ | eV ↔ kcal/mol ↔ kJ/mol |
| ~normalize_energy_unit~ | Alias normalization |
| ~suite_pins~ | Shared rgpkgs package pins via hub |

Submodules (~chemparseplot.parse~, ~chemparseplot.plot~) remain available for
advanced use; plot stacks may need optional deps until fully on-demand.

* Config files (suite-wide)

- ~/~/.config/rgpkgs/config.toml~
- Project ~rgpkgs.toml~
- Env ~RGPKGS_LOCK~, ~RGPKGS_CONFIG~, ~RGPKGS_FORCE_UV~, ~RGPKGS_AUTO_DEPS~

Hub has *no* runtime feature extras. Do not ~pip install "rgpycrumbs[analysis]"~.
